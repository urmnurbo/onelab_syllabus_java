# onelab_syllabus_java
## Syllabus in OneLab for Java EE course:**
```
* Spring - Инверсия зависимости, бины, как работает под капотом Spring
* Аспектно-ориентированное программирование
* Подключение и работа с БД с использованием ORM Hibernate, Spring Data
* Test-driven development, TDD, как писать Unit тесты
* Spring Java Message Service. Использование Kafka в проекте
* Сможете создать веб-приложений с использованием Spring, разработке REST-сервисов, а также об интеграции с библиотекой Netflix.
* Как писать качественный код который НЕ пахнет
* Camunda platform.
```
